Math support in Cyblog

Cyblog supports math markup. Simply put your expression in a code span / block.

If you put the expression in a code span, it will be rendered inline.

For example, $\large\frac{4}{5}$ will result in 45\large\frac{4}{5}54​.

If you put it in a code block, it will be rendered in display mode.

$$
\large
T(n) =  n![{2n \choose n} - {2n \choose n+1}]
$$

will become:

T(n)=n![(2nn)−(2nn+1)] \large T(n) = n![{2n \choose n} - {2n \choose n+1}] T(n)=n![(n2n​)−(n+12n​)]

You can escape a math expression by prefixing it with a backslash.

Cyblog uses the excellent KaTeX library to render math.
